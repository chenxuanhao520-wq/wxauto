## 🚀 **使用方法**

### **开发环境启动**

#### 后端开发
```bash
cd backend
pip install -r requirements.txt
python3 main.py
```
**访问**: http://localhost:8000

#### 前端开发
```bash
cd frontend
npm install
npm run dev
```
**访问**: http://localhost:3000

#### 客户端开发
```bash
cd client
python3 cloud_client.py
```

### **生产环境部署**

#### 一键部署
```bash
cd docker
./deploy.sh
```

#### 手动部署
```bash
cd docker
docker-compose up -d
```
