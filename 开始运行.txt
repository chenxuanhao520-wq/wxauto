Q: 必须安装 Python 吗？
A: 是的，这是必需的。推荐 Python 3.10 或 3.11。

Q: 必须配置 API Key 吗？
A: 测试模式不需要。真实使用需要至少一个大模型 API Key。

Q: 推荐哪个大模型？
A: DeepSeek（便宜，¥0.1/百万tokens）或 OpenAI（质量好）

Q: 需要真实微信吗？
A: 测试模式不需要。真实模式需要 Windows + PC 微信。

Q: 支持语音和图片吗？
A: 支持！需要额外安装：
   pip install funasr paddleocr paddlepaddle

Q: 成本多少？
A: 基础版约 ¥100/月，主要是大模型 API 费用。

Q: 安装需要多久？
A: 基础版 5-10 分钟，完整版 30 分钟（含模型下载）。
